Git - система контроля версий. Основная идея использования заключается в том, чтобы иметь возможность откатиться к старой, стабильной, работающей версии чего бы то ни было, если что-то пойдет не так. В качестве приятного бонуса идет возможность отследить историю изменения проекта.
